# SD-BACK-03 · Merge Special Days into Calendar API

**Timestamp**: 20:17:28 Local Time  
**Author**: Ralph  

## Why
Enable the frontend to receive all necessary data (events, special days, and tag configurations) in a single request to the main calendar endpoint, simplifying the rendering logic.

## What Changed
- `backend/main.py`:
    - Updated `get_calendar` endpoint:
        - Loads `special_days.json` and `tags_config.json`.
        - Indexes special days by date for efficient merging.
        - Injects `special` metadata into each day object in the calendar grid.
        - Adds `special_days` list and `tags_config` object to the final JSON response.
