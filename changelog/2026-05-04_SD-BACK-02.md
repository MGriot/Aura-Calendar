# SD-BACK-02 · Implement API endpoints for Special Days and Tags

**Timestamp**: 20:16:18 Local Time  
**Author**: Ralph  

## Why
Expose CRUD interfaces for the new "Twin Calendar" feature so the frontend can manage tags and assign them to specific dates.

## What Changed
- `backend/main.py`:
    - Added `TagConfigItem`, `TagsConfig`, `SpecialDay`, and `AssignRequest` Pydantic models.
    - Implemented `load_tags_config`, `save_tags_config`, `load_special_days`, and `save_special_days` for persistence.
    - Added endpoints:
        - `GET /api/tags`: Returns tag configuration.
        - `POST /api/tags`: Updates tag configuration.
        - `GET /api/special-days`: Returns all tagged days.
        - `POST /api/special-days/assign`: Bulk assigns tags/locations to multiple dates.
